## 第一部分 实验准备

### 1. 实验目标

1）掌握逻辑回归的基本概念。

2）掌握机器学习的一般流程。

3）掌握如何使用MindSpore进行逻辑回归实验。

4）掌握如何使用华为云ModelArts Notebook上传数据、执行Python代码。

### 2. 实验内容

取Iris数据集中的前两类数据（共100条），进行部分数据可视化，并将数据集分为训练集与测试集，使用`nn.Dense(4,1)`算子与`SigmoidCrossEntropyWithLogits()`算子进行建模并进行模型训练，最后对模型作评估
